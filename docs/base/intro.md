# 认识社区

Zaofan Hub 是一个技术性造反爱好者社区。

## MVV - 使命、愿景、价值观

- **`使命`**: 研究一切技术性造反的可能
- **`愿景`**: 成为有影响力的技术性造反爱好者社区
- **`价值观`**: 隐私自主、开放协作、代码实践

## 具体要做的事情

- 技术与文化传播 - 隐写、私有网络、加密、去中心化、信息传递、P2P
- [Whisper - 文本隐写项目](https://whisper.zaofan.org)
- [社区文化 - 爱好者聚集地、技术交流、公共讨论](https://forum.zaofan.org)
